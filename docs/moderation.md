# モデレーション方針

## 原則

1. プライバシーの絶対的保護
- 基準：個人情報や特定個人を特定できる情報（心理状態、治療内容、個人識別情報など）の公開や共有は、ユーザー自身のものを含めて禁止します。ただし、ユーザーが自ら進んで開示する場合は、開示者の自由を尊重しますが、他者が第三者の情報を開示することは固く禁じます。コミュニティ内の会話は外部に漏らさないことが期待されます。
- 根拠：個人の自己決定権を尊重するリベラルな考え方に基づき、プライバシーの侵害が運営者と他者に与えるリスクを最小化しつつ、ユーザーが自分の情報をどのように扱うかは自由に選択できるようにします。法的にも、GDPRやその他のプライバシー保護法に準拠しています。

2. 支え合いと尊重を基本とした行動の推奨
- 基準：他者を尊重し、建設的な対話を重視することが求められます。他者を攻撃したり、精神的に追い詰める行動は禁止されますが、批判や意見の表明自体は自由です。重要なのは、他者に対して感情的なダメージを与えるのではなく、尊重し合う姿勢を持つことです。
- 根拠：リベラルな自由主義に基づき、ユーザーが意見を表明する自由を守りつつ、その自由が他者を傷つけない範囲にとどまるようにします。これにより、個人の自由とコミュニティ全体の調和が保たれます。

3. 自己破壊的行動に関する柔軟な対応
- 基準：自殺念慮や自己傷害に関する具体的な内容や助言は慎重に扱いますが、これを完全に禁止するのではなく、ユーザーがそのような内容に触れる場合、コミュニティ運営者は適切なサポートリソースを提示します。コンテンツの削除は最終手段とし、まずは支援や対話を促進する方向で対応します。
- 根拠：非干渉の倫理と支援的介入のバランスに基づき、ユーザーが自己表現を制限されることなく自由に発言できる一方で、他者に対して害を与える可能性がある場合には、運営者が適切に介入します。このアプローチは、運営者が法的リスクを回避しつつも、ユーザーの自由を最大限尊重するものです。

4. 過度な依存の緩やかな管理
- 基準：他者に対する依存やサポート要求がコミュニティ全体に過度な負担をかける場合、運営者はリソース管理を考慮し、適度に介入します。ただし、最初の段階では制限せず、利用者間でのサポートが可能な限り自発的に行われることを奨励します。リソースが枯渇するリスクが出てきた場合にのみ、運営者が柔軟に対応します。
- 根拠：自律と相互支援に基づき、コミュニティの自然なサポートネットワークを促進し、個々のユーザーの自律的な行動を尊重します。過度な依存がコミュニティ全体に悪影響を与える場合は、功利主義的な視点でリソースの配分を調整します。

5. コミュニティ全体の健全性維持のための最小限の制約
- 基準：スパム、荒らし行為、過剰な自己宣伝、挑発的な行為は、コミュニティの秩序を乱す場合に限り対応します。これらの行為がコミュニティの運営や他者のメンタルヘルスに直接的な悪影響を与えない限り、ユーザーの自由な発言や行動は最大限尊重されます。
- 根拠：運営者は、コミュニティの秩序と健全性を守るために必要最小限の介入を行うべきであり、過剰な制限を避けます。このアプローチは、リベラルな自由主義と個人の自律性を基盤にしており、個人の表現の自由が守られる環境を作り出します。


## 思想

1. リベラリズムと個人の自律性
このモデレーション基準の根底には、リベラリズムがあり、個人の自律性と自由が最優先されます。ユーザーが自己決定権を持ち、できる限りの自由な発言や行動を行うことができる一方で、その自由が他者に害を与えないことが前提です。運営者の介入は最小限に抑えられ、ユーザーが自由に自己表現できる環境が保たれます。

2. ケア倫理と相互支援
コミュニティ内では、ケア倫理に基づいて他者への配慮と支援が奨励されます。これは、コミュニティ全体の幸福と安全を保つために、ユーザーが互いに支え合う姿勢を基本としています。ユーザーが自由に発言しながらも、相互に支え合う文化が形成されることが、健全なコミュニティ運営につながります。

3. プロセスの透明性と最小限の干渉
手続き的正義に基づき、モデレーションの決定は透明で、公平なプロセスによって行われます。ユーザーに対して理由が明確に示され、すべての人に対して同じ基準が適用されることで、信頼性と公正さが保たれます。また、運営者の干渉は最小限にとどめ、ユーザーの自由な行動をできる限り尊重します。

4. 法的保護
プライバシー保護と自己破壊的行動への対応は、法的観点からも重要です。ユーザーの個人情報を適切に保護し、運営者が法的責任を負うリスクを避けるため、プライバシーや自殺念慮に関する基準は厳密に運用されます。ただし、法的な制約が自由を過度に抑えないように注意し、法的リスクを最小化しつつ、最大限の自由を提供します。
